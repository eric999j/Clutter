# 實驗功能
chrome://flags/  
1. Enable sharing page via OR Code: 網頁右鍵變QR Code分享 
2. Parallel downloading: 加快下載速度  
3. Force Dark Mode for Web Contents: 全面黑化 不推    
4. TLS 1.3 hardening for local anchors: 增強安全，減少延遲  
指令輸入：defaults write com.google.Chrome NSRequiresAquaSystemAppearance -bool YES  只有外誆黑色，比較好看
